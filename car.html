Cars Dataset Analysis

I am working with a dataset containing information about cars. My first task is to perform data cleaning by identifying columns with null values and filling those null values with the mean value of the respective column. Next, I need to explore the dataset by checking the different types of car manufacturers (Make) present and their corresponding counts using the value_counts() method. After that, I need to filter the dataset to include only records where the car's origin is from Asia or Europe using the isin() method. Additionally, I need to remove all records where the car's weight exceeds 4000 units. Finally, I need to apply a function to the 'MPG_City' column using the apply() method, which will increase all values in that column by 3 units. This process will help me clean, explore, and manipulate the car dataset effectively, allowing me to gain insights and prepare the data for further analysis or modeling.
